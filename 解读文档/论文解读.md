## 一、论文概述

### 1.1 研究的问题

这篇论文研究的核心问题是：**如何为扩展目标设计雷达发射波形，使检测性能在目标不确定性下尽可能稳定？**

传统方法只关心"平均性能好不好"，而本文提出的 PRD 准则同时关心"性能波动大不大"。

### 1.2 核心贡献

| 贡献 | 内容 |
|:---|:---|
| 新准则 | 提出 PRD（概率鲁棒检测）波形设计准则 |
| 高SINR算法 | 基于 Dinkelbach 算法，可获得**全局最优** |
| 低SINR算法 | 基于 Dinkelbach + MM 技术，可获得**次优解** |
| 理论证明 | 证明了两种情况下算法的收敛性 |

### 1.3 对比方法

本文对比的基线方法是 Chen & Vaidyanathan (2009) 的文献[17]，该方法只最大化输出SINR的数学期望 E[γ]。

## 二、基础概念

### 2.1 什么是扩展目标？

**点目标**：目标尺寸远小于雷达距离分辨率，在雷达屏幕上只占一个距离单元。例如远距离的小鸟。

**扩展目标**：目标尺寸超过雷达距离分辨率，占据**多个**距离单元。例如战斗机（15米）或客机（38米）在高分辨率雷达下就是扩展目标。

区别：

- 点目标的回波 = 发射波形的缩放版本
- 扩展目标的回波 = 发射波形与目标冲激响应（TIR）的**卷积**

### 2.2 什么是目标冲激响应（TIR）？

TIR 描述了目标在各个距离单元上的反射特性。用向量 t = [t(1), t(2), ..., t(Q)] 表示，其中 Q 是目标占据的距离单元数。

- t(q) 的幅度：第 q 个距离单元的反射强度
- t(q) 的相位：该处的散射相位

**关键问题**：TIR 高度依赖于目标的姿态角、视线方向等，在实际检测前**无法精确获知**。

### 2.3 什么是 SINR？

SINR（Signal-to-Interference-plus-Noise Ratio，信干噪比）是衡量信号质量的核心指标：

```
SINR = 有用信号功率 / (干扰 + 噪声功率)
```

SINR 越大，信号越"干净"，越容易检测目标。

### 2.4 什么是 PAR 约束？

PAR（Peak-to-Average Ratio，峰均比）约束是雷达硬件的实际限制。雷达发射机的功放通常工作在饱和区，不允许波形幅度变化太大。

PAR 约束表示为：波形总能量为 1，且每个采样点的功率不超过 ρ。

- ρ = 1：能量约束（EC），对幅度无限制
- ρ = 1/L：恒模约束（CMC），所有采样点功率必须相等（如相位编码波形）

### 2.5 什么是半正定松弛（SDR）？

原始波形设计问题是NP-hard的（关于波形向量 s 的高维非凸优化）。SDR 是一种常用的近似技术：

1. 定义波形协方差矩阵 $Rs = s·s^H$（秩为1的矩阵）
2. 去掉"Rs必须秩为1"的约束，只保留 Rs 半正定
3. 在更大的可行域上求解（凸优化）
4. 从优化后的 Rs 合成实际波形 s

### 2.6 什么是 Dinkelbach 算法？

Dinkelbach 算法专门用于求解**分式优化问题**（目标函数是两个函数的比值）：

```
max  a(x) / b(x)
```

核心思想：将分式问题转化为一系列**非分式**子问题迭代求解。

每次迭代求解：max a(x) - f^(k-1) · b(x)，其中 f^(k-1) 是上一轮的目标值。

当目标函数是拟凸的时候，Dinkelbach 算法保证收敛到**全局最优**。

### 2.7 什么是 MM 技术？

MM（Minorization-Maximization）是处理非凸优化问题的通用框架：

1. 在当前点构造原目标函数的一个**下界函数**（minorizer）
2. 最大化这个下界函数得到新的点
3. 重复，直到收敛

关键性质：每次迭代的目标值**单调不减**，保证收敛。

## 三、信号模型

### 3.1 接收信号模型

雷达发射离散波形 s = [s(1), s(2), ..., s(L)]，经过扩展目标反射后，接收信号为：

```
y = T·s + n        ...(1)
```

其中：

- T 是由 TIR 构成的卷积矩阵（大小 (L+Q-1)×L）
- n 是接收端的干扰噪声，n ~ CN(0, Rn)

等价地，可以改写为：

```
y = S·t + n        ...(2)
```

其中 S = [J₀·s, J₁·s, ..., J_{Q-1}·s] 是由波形构造的矩阵，Jq 是移位矩阵。

### 3.2 输出 SINR

经过最优滤波器处理后，输出 SINR 为：

```
γ(s) = t^H · S^H · Rn^{-1} · S · t     ...(3)
```

定义矩阵 M = S^H · Rn^{-1} · S（大小 Q×Q），则：

```
γ(s) = t^H · M · t
```

M 矩阵是波形 s 的函数，编码了"波形对各个距离单元的探测能力"。

### 3.3 随机 TIR 模型

由于 TIR 不确定，采用随机模型：

```
t ~ CN(t₀, Rt)
```

- t₀：TIR 的均值（先验知识，"大概长这样"）
- Rt：TIR 的协方差矩阵（不确定度，"可能偏离多少"）

在此模型下，γ(s) = t^H·M·t 也变成随机变量。

### 3.4 SINR 的统计特性（Proposition 1）

论文推导了 γ(s) 的封闭形式统计特性：

**分布**：γ(s) 是若干独立非中心卡方随机变量的加权和。

**期望**：

```
E[γ] = tr(Rt·M) + t₀^H·M·t₀
```

含义：第一项是 TIR 不确定性带来的平均 SINR 贡献；第二项是已知 TIR 均值的确定性贡献。

**方差**：

```
V[γ] = tr(Rt·M·Rt·M) + 2·t₀^H·M·Rt·M·t₀
```

含义：方差反映了 SINR 因 TIR 随机性带来的波动程度。

## 四、PRD 波形设计准则

### 4.1 设计思路的起点

检测概率与 SINR 的关系（Marcum-Q 函数）：

```
Pd = MarcumQ(√(2γ), √(-2ln(Pfa)))    ...(4)
```

检测概率曲线呈"S"形——SINR 从小到大，检测概率快速从 0 增长到 1。

当 SINR 足够大时（如 13 dB），检测概率已经接近 99.7%，再增大 SINR 意义不大。

### 4.2 核心问题：TIR 随机导致检测概率不稳定

如果 TIR 是确定的，我们只需最大化 γ 即可。但 TIR 是随机的，γ 也随机——有时大有时小。

理想目标：最大化 "γ 超过阈值 γ₀ 的概率"：

```
max  Pr{γ(s) ≥ γ₀}     ...(5)
  s
```

但 Pr{γ ≥ γ₀} 的表达式极其复杂（涉及多个非中心卡方分布的积分），无法直接优化。

### 4.3 代理函数 f(s)——PRD 准则

论文提出用以下**代理函数**来替代：

```
f(s) = (E[γ] - γ₀) / D[γ]     ...(7)
```

其中 D[γ] = √V[γ] 是 γ 的标准差。

**这个准则的直觉含义**：

- 分子 E[γ] - γ₀：SINR 期望与阈值的距离（越大越好）
- 分母 D[γ]：SINR 的波动程度（越小越好，代表越稳定）
- f 值越大，"平均性能好且波动小"，检测越稳定

### 4.4 几何解释

论文从两个角度解释了为什么 f(s) 是好的代理：

**高SINR情况**（E[γ] ≥ γ₀）：

此时 γ 的分布主体在 γ₀ 右侧。要让 Pr{γ ≥ γ₀} 更大：

- 需要 E[γ] 尽量远离 γ₀（分子大）
- 需要 D[γ] 尽量小（分布集中，不会"漏"到 γ₀ 左边）

→ 最大化 f(s) 正好同时实现这两点。

**低SINR情况**（E[γ] < γ₀）：

此时 γ 的分布主体在 γ₀ 左侧。要让 Pr{γ ≥ γ₀} 尽量大：

- 需要 E[γ] 尽量靠近 γ₀（缩小与阈值的差距）
- 需要 D[γ] 尽量大（分布越散，"漏"到 γ₀ 右边的就越多）

此时 f(s) = (负数) / D[γ]，分子是负数。要最大化 f（让它最接近0），需要分子绝对值小（E[γ]接近γ₀）且分母大（D[γ]大）——同样符合直觉。

### 4.5 极限解释

当 TIR 维度 Q 趋向无穷时，由中心极限定理，γ 趋近高斯分布。此时：

```
Pr{γ ≥ γ₀} = 1 - Φ((γ₀ - E[γ]) / D[γ]) = Φ(f(s))
```

其中 Φ 是标准正态分布的CDF。由于 Φ 单调递增，最大化 Pr{γ ≥ γ₀} 等价于最大化 f(s)。

### 4.6 与文献[17]的关系

文献[17]的准则：max E[γ]（只最大化期望）。

PRD 准则是[17]的**推广**：

- 当 V[γ] 为常数且 t₀ = 0 时，PRD 退化为 [17]
- PRD 额外考虑了 SINR 分布的**二阶统计量**（方差），因此更加鲁棒

### 4.7 完整优化问题

```
P: max  f(s) = (E[γ] - γ₀) / D[γ]
   s.t.  ||s||² = 1
         |s(i)|² ≤ ρ,  i = 1,...,L     ...(10)
```

这是一个 NP-hard 问题，原因：

1. PAR 约束非凸
2. f(s) 是关于 s 的非线性分式
3. D[γ] 是关于 s 的四次函数的平方根

## 五、优化算法设计

### 5.1 半正定松弛

首先将问题转化为对波形协方差矩阵 Rs = s·s^H 的优化。

M 矩阵关于 Rs 是**线性**的：M(m,n) = tr(F{m,n} · Rs)，其中 F{m,n} = Jm^H · Rn^{-1} · Jn 是可预计算的常数矩阵。

因此 E[γ] 和 V[γ] 都可以表示为 Rs 的函数。

松弛后的问题：

```
P̃: max  (E[γ(Rs)] - γ₀) / D[γ(Rs)]
    Rs
    s.t.  tr(Rs) = 1
          Rs(i,i) ≤ ρ,  i = 1,...,L
          Rs ≥ 0  (半正定)               ...(11)
```

### 5.2 初始化（公式23）

Dinkelbach 算法需要可行初始点。论文通过求解以下线性 SDP 获得：

```
P̃₀: max  E[γ] - γ₀ = tr(Z_num · Rs) - γ₀
     Rs
     s.t.  tr(Rs) = 1, Rs(i,i) ≤ ρ, Rs ≥ 0
```

初始化目标值 V(P̃₀) 的正负决定了 SINR 情况：

- V(P̃₀) ≥ 0 → 高SINR → 调用 Algorithm 1
- V(P̃₀) < 0 → 低SINR → 调用 Algorithm 2

### 5.3 Algorithm 1：高SINR（Dinkelbach + SDP）

**高SINR条件**：存在可行 Rs 使得 E[γ] ≥ γ₀。

**关键定理（Proposition 2）**：在高SINR条件下，P̃ 是**拟凸**优化问题。

Dinkelbach 迭代：每次求解

```
max  (E[γ] - γ₀) - f^(k-1) · √V[γ]
 Rs
s.t.  tr(Rs)=1, Rs(i,i)≤ρ, Rs≥0
      E[γ] - γ₀ ≥ 0                      ...(14)
```

其中 f^(k-1) 是上一轮的 f 值。

由于 √V[γ] 关于 Rs 是凸的，而 E[γ]-γ₀ 是线性的，且 f^(k-1) ≥ 0，所以每个子问题是**凸优化**（最大化凹函数在凸集上），可以用 SDP 高效求解。

**收敛性**：Dinkelbach 算法对拟凸问题保证全局最优。

### 5.4 Algorithm 2：低SINR（Dinkelbach + MM + SDP）

**低SINR条件**：所有可行 Rs 都有 E[γ] < γ₀。

此时 f^(k-1) < 0，子问题变成：

```
max  (E[γ] - γ₀) + |f^(k-1)| · √V[γ]
```

加号前面是线性的，后面 |f^(k-1)|·√V[γ] 是**凸的** → 最大化凸函数 → **非凸**！

**解决办法：MM 技术**

在当前点 M^(k) 处，用 √V[γ] 的线性下界（支撑超平面）代替：

```
√V[M] ≥ √V[M^(k)] + tr(G · (M - M^(k)))
```

其中 G 是 √V 在 M^(k) 处的梯度（论文公式17）。

替换后，每个 MM 子问题变成线性 SDP：

```
max  tr(C · M)                           ...(18)
 Rs
s.t.  tr(Rs)=1, Rs(i,i)≤ρ, Rs≥0

其中 C = (Rt + t₀·t₀^H) - f^(k-1) · G
```

**收敛性**：MM 技术保证目标值**单调不减**且有上界 → 收敛。但只能保证收敛到**局部最优**（次优解）。

### 5.5 Algorithm 3：完整流程

```
步骤1: 预计算系数矩阵 F{m,n}、Z_num、Z_B
步骤2: 求解初始化问题 P̃₀
步骤3: 判断高/低 SINR
        若 V(P̃₀) ≥ 0 → Algorithm 1
        若 V(P̃₀) < 0 → Algorithm 2
步骤4: 从优化的 Rs 合成波形 s*
```

### 5.6 波形合成

从优化后的 Rs 合成实际波形 s*：

- 若 Rs 接近秩-1：直接特征分解提取主特征向量
- 否则：随机化方案——生成 N=1000 个候选波形，选择 f 值最大的

对于 CMC（恒模约束），合成方法是：提取相位，设置恒模幅度：

```
s* = exp(j·angle(v)) / √L
```

## 六、仿真实验与结果分析

### 6.1 实验设置

| 参数 | 值 |
|:---|:---|
| 波形采样点数 L | 60 |
| 脉冲宽度 | 0.6 μs |
| 采样率 | 100 MHz |
| 距离分辨率 | 1.5 m/距离单元 |
| 噪声模型 | 白噪声 Rn = σ²·I |
| 虚警率 Pfa | 10^{-6} |
| 期望SINR阈值 γ₀ | 15 dB |
| 收敛阈值 | 10^{-4} |
| 波形合成样本数 N | 1000 |

**两个测试目标**：

| | 目标1（战斗机） | 目标2（客机） |
|:---|:---|:---|
| 长度 | 15 m | 38 m |
| 距离单元数 Q | 10 | 25 |
| t₀ 特点 | 非均匀，有明显主瓣 | 均匀，t₀ = 0.1·1 |
| Rt 特点 | 一个频率分量 | 三个频率分量 |

### 6.2 实验1：收敛性验证（Fig.3）

**目的**：验证 Dinkelbach 迭代是否快速收敛。

**衡量**：辅助目标值 (E[γ]-γ₀) - f^(k-1)·√V[γ] 应趋向 0。

**结果**：

- 高SINR：3~7 次迭代收敛到 0
- 低SINR：初始化即接近最优（辅助目标值几乎为 0）

**结论**：算法收敛性良好，与理论分析一致。

### 6.3 实验2：波形合成验证（Fig.4 + Fig.5）

**Fig.4 — 波形功率曲线**：

- 目的：验证合成波形满足 PAR 约束
- 结果：ρ=1/20 时最大功率 ≤ 0.05；CMC 时功率恒定

**Fig.5 — 合成损失**：

- 目的：评估 SDR 松弛引起的性能损失
- 结果：最大损失 < 0.1，ρ 对性能影响很小

### 6.4 实验3：SINR 分布对比（Table I + Fig.6/7）

**这是论文最核心的对比实验。**

**做法**：用两种方法设计波形，随机生成 1000 个 TIR 样本，统计 SINR 分布。

**关键发现**：

**高SINR情况（σ² 小）**：

- [17] 方法：E[γ] 和 D[γ] 以**相同比例**增长
  - 例：E=285, D=278 → f = 0.91
- PRD 方法：E[γ] 增长的同时 D[γ] 被**有效抑制**
  - 例：E=130, D=44 → f = 2.28
- PRD 的 f 值比[17]高 **150%**

**低SINR情况（σ² 大）**：

- 两种方法表现**完全一致**
- PRD 退化为 [17]

**物理解释**：

- [17] 把所有能量集中在"最赚"的频点 → 贪婪策略，期望高但波动大
- PRD 把能量分散到多个频点 → 保守策略，牺牲一些期望换取稳定性

### 6.5 实验4：波形PSD对比（Fig.8/9）

**目的**：从频域解释两种方法的差异。

**关键发现**：

- **[17] 方法**：波形PSD始终在目标PSD最大峰处形成**单一尖峰**，不随 σ² 变化
- **PRD 方法（高SINR）**：波形PSD覆盖**目标PSD的整体多峰结构**，能量分散
- **PRD 方法（低SINR）**：与 [17] 趋于一致

**直觉**：PRD 波形"模仿"目标的频域特征，即使某个频点的 TIR 偏离预期，其他频点仍能保证检测——这就是"鲁棒性"的来源。

### 6.6 实验5：稳定检测概率（Fig.10）

**目的**：直接回答最终问题——"检测概率提升了多少？"

**稳定检测概率定义**：

```
Psd = (γ ≥ γ₀ 的次数) / (总蒙特卡洛次数)
```

**做法**：扫描 σ²，对每个噪声水平计算两种方法的 Psd。

**结果**：

| σ² 区域 | PRD vs [17] |
|:---|:---|
| 低SINR（σ² 大） | 两者重合，Psd 都很低 |
| 过渡区 | PRD 的 Psd 比 [17] 高 6%~20% |
| 高SINR（σ² 小） | PRD 更快趋近 Psd = 1 |

**结论**：PRD 波形在**同样的噪声条件下**有更高的概率成功检测目标。

## 七、总结

### 7.1 论文的核心思想（一句话）

> **不要只追求平均最优，还要保证稳定可靠——通过同时优化 SINR 的期望和方差，实现更高的"达标率"。**

### 7.2 方法对照表

| 维度 | 文献[17] | 本文PRD |
|:---|:---|:---|
| 优化目标 | max E[γ] | max (E[γ]-γ₀) / D[γ] |
| 统计量 | 一阶（期望） | 一、二阶（期望+方差） |
| 频域策略 | 集中能量（贪婪） | 分散能量（鲁棒） |
| 高SINR性能 | E[γ]高但D[γ]也高 | E[γ]略低但D[γ]显著低 |
| 低SINR性能 | 基准 | 与基准一致 |

### 7.3 论文的局限与展望

1. 仅考虑**信号无关干扰**，未考虑信号相关杂波
2. 未联合设计接收滤波器（只设计发射波形）
3. 低SINR只能获得次优解，缺乏全局最优保证
4. 未考虑相似性约束和频谱兼容性约束
