# 实验5：稳定检测概率曲线（Fig.10）详细讲解

## 从零开始理解这个实验

---

### 第一步：理解"检测"是什么意思

雷达的基本任务就是：**判断前方有没有目标**。

雷达发射一个波形 s，如果前方有目标，信号经过目标反射后被接收；如果没有目标，接收到的只有噪声。雷达需要根据接收信号做出判断：**有目标** or **没有目标**。

这个判断依赖于一个关键数值：**输出信噪比 γ（SINR）**。

- γ 很大 → 信号远强于噪声 → 有信心判断"有目标"
- γ 很小 → 信号淹没在噪声中 → 难以判断

雷达工程师会设定一个**阈值 γ₀**（论文中设为 15 dB），规则很简单：

```
如果 γ ≥ γ₀ → 判断"检测到目标"（成功）
如果 γ < γ₀ → 判断"没检测到"  （失败）
```

---

### 第二步：理解为什么 γ 是"随机"的

在传统雷达中，目标的反射特性（TIR）是已知的，γ 是一个确定的值。但对于**扩展目标**，TIR t 在检测前无法精确知道——它与目标姿态、视角等因素有关，每次照射都可能不同。

论文用随机模型描述这种不确定性：

```
t ~ CN(t₀, Rt)
```

- t₀ 是 TIR 的均值（"大概长这样"）
- Rt 是 TIR 的协方差矩阵（"可能在这个范围内波动"）

因此，γ 也变成了一个**随机变量**——同一个波形 s，面对不同的TIR实现，得到的 SINR 不同。有时候运气好（TIR接近预期），γ 很大，检测成功；有时候运气差（TIR偏离预期），γ 很小，检测失败。

---

### 第三步：理解"稳定检测概率 Psd"

既然每次检测的 γ 不同，我们就问一个概率问题：

> **在所有可能的TIR中，有多大比例能让我成功检测？**

这就是**稳定检测概率**的含义：

```
Psd = 成功检测的次数 / 总试验次数 = N_EX / N_TO
```

- N_TO = 总试验次数（比如 1000 次）
- N_EX = 其中 γ ≥ γ₀ 的次数

**Psd = 0.8** 意味着：用这个波形检测 1000 次不同的TIR，大约 800 次能成功检测。

**Psd = 0.3** 意味着：只有 300 次能成功，700 次失败——太不稳定了。

---

### 第四步：噪声 σ² 如何影响结果

σ² 是**干扰（噪声）功率**。它的影响很直观：

| σ² 大小 | 物理含义 | 对 γ 的影响 | 对 Psd 的影响 |
|:---|:---|:---|:---|
| σ² 很大（如 20 dB） | 噪声很强 | γ 普遍很小 | Psd ≈ 0（几乎全检测失败） |
| σ² 适中（如 0 dB） | 噪声中等 | γ 在阈值附近 | Psd 在 0~1 之间 |
| σ² 很小（如 −10 dB） | 噪声很弱 | γ 普遍很大 | Psd ≈ 1（几乎全检测成功） |

所以 **σ² 从大变小的过程**，就是 Psd 从 0 上升到 1 的过程。

---

### 第五步：为什么要对比两种方法？

论文的核心主张是：**PRD 准则比 [17] 准则更"稳定"**。

两者在低SINR下都检测不到（Psd ≈ 0），在极高SINR下都能检测到（Psd ≈ 1）。关键区别在**过渡区间**——谁的 Psd**更快**从 0 上升到 1。

```
Psd
1.0 ┤                              ╭──── PRD 先到 1
    │                          ╭───╯
    │                      ╭───╯
    │                  ╭───╯    ╭─── [17] 后到 1
    │              ╭───╯    ╭───╯
    │          ╭───╯    ╭───╯
    │      ╭───╯    ╭───╯
0.0 ┤──────╯────────╯
    └────────────────────────────────→ σ² 减小（SINR 变好）
```

PRD 曲线在中间区域高于 [17]，意味着在**同样的噪声条件下**，PRD 设计的波形有**更高的概率**成功检测目标。

---

### 第六步：实验的完整实现流程

用伪代码描述整个实验：

```
# 实验参数
γ₀ = 10^(15/10)          # SINR阈值 = 15 dB
ρ = 1/L                   # CMC 恒模约束
N_MC = 1000               # 蒙特卡洛试验次数

# 目标1: 扫描不同的 σ² 值
σ²_list = [-10, -8, -6, -4, -2, 0, 2, 4, 6] dB
# 目标2: 另一组 σ² 值
σ²_list = [-10, -5, 0, 5, 10, 15, 20] dB

对每个 σ²:
  ① 构造干扰逆矩阵: Rn_inv = (1/σ²) * I

  ② 用 PRD 准则设计波形 s_prd
     → PRD_design(t0, Rt, Rn_inv, ρ, L, Q, γ₀)

  ③ 用 [17] 准则设计波形 s_ref
     → baseline_metric17(t0, Rt, Rn_inv, ρ, L, Q, γ₀)

  ④ 计算两种波形对应的 M 矩阵
     M_prd = S_prd^H * Rn_inv * S_prd  （通过 compute_M）
     M_ref = S_ref^H * Rn_inv * S_ref

  ⑤ 蒙特卡洛仿真：重复 1000 次
     对 i = 1 到 1000:
       随机生成一个 TIR: t = t₀ + Rt_chol * 随机复高斯向量
       计算 γ_prd(i) = t^H * M_prd * t
       计算 γ_ref(i) = t^H * M_ref * t

  ⑥ 统计成功率
     Psd_prd = count(γ_prd ≥ γ₀) / 1000
     Psd_ref = count(γ_ref ≥ γ₀) / 1000

# 绘图：横轴 σ²(dB)，纵轴 Psd
# 蓝色曲线 = PRD，红色曲线 = [17]
```

> [!TIP]
> 注意：步骤⑤和 exp3a_sinr_hist.m 的蒙特卡洛完全一样（都是随机生成TIR计算γ），唯一区别是 exp3a 画直方图，而 Fig.10 只统计"超过阈值的比例"。

### 第七步：与前面实验的关系

Fig.10 其实是对 Fig.6/7 结果的**进一步提炼**：

```
Fig.6/7 的直方图              →  Fig.10 的 Psd
                                          
  ┌─────────────┐              只关注直方图中
  │  ▓▓▓ ▓▓▓    │γ₀           γ₀ 右侧的面积占比
  │ ▓▓▓▓▓▓▓▓▓   │──→          
  │▓▓▓▓▓▓▓▓▓▓▓  │             蓝色面积/(蓝色总面积) = Psd_prd
  └─────────────┘             红色面积/(红色总面积) = Psd_ref
```

在 Fig.6(a) 中（高SINR），蓝色（PRD）的直方图集中在 γ₀ 附近且方差小，绝大部分都在 γ₀ 右侧 → Psd 高。红色（[17]）虽然均值更大但分布很散，有一部分落在 γ₀ 左侧 → Psd 相对低。

这就是 PRD "概率鲁棒"的含义：**不追求最高的期望 SINR，而是追求最高比例的"达标率"**。

---

### 总结

| 概念 | 含义 |
|:---|:---|
| γ (SINR) | 信号质量的数值度量 |
| γ₀ | 检测阈值（15 dB），超过它算"检测成功" |
| σ² | 噪声功率，越大检测越难 |
| TIR t | 目标反射特性，随机变化 |
| 蒙特卡洛 | 随机抽样 1000 次模拟不同TIR情况 |
| N_EX | 1000 次中 γ ≥ γ₀ 的次数 |
| Psd | N_EX / 1000 ，稳定检测概率 |
| Fig.10 | 画 Psd 随 σ² 变化的曲线，比较两种方法 |
