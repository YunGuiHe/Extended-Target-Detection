## 二、逐实验详细分析

### 实验1：收敛性验证（Fig.3）

**目的**：验证本文提出的 Algorithm 3（Dinkelbach迭代框架）是否能够快速收敛。论文在理论上证明了高SINR情况下的全局最优性和低SINR情况下的收敛性，此实验从**数值角度**验证这一理论结果。

**做法**：记录每次 Dinkelbach 迭代的辅助目标值（应收敛到 0），绘制4个子图（目标1/2 × 高/低SINR）。

**衡量指标**：辅助目标值 = (E[γ] − γ₀) − f⁽ᵏ⁻¹⁾·√V[γ]，该值趋近于 0 说明算法收敛。

**期望结果**：高SINR约 3~5 次迭代收敛；低SINR初始化即接近最优（辅助目标值接近 0）。

**复现必要性**：⭐⭐⭐ **建议复现**。这是验证算法正确性的基础实验，能直观确认代码实现是否正确。

### 实验2a：合成波形功率（Fig.4）

**目的**：验证从优化的协方差矩阵 Rs 合成出的实际波形 s* 是否**满足PAR约束**。这是工程实现中的关键约束——雷达发射机功放通常工作在饱和区，不允许波形幅度深度调制。

**做法**：绘制波形每个采样点的功率 |s*(i)|²，检查是否不超过 ρ。

**衡量指标**：max |s*(i)|² ≤ ρ。

**期望结果**：ρ=1/20 时最大功率 ≤ 0.05；ρ=1/L (CMC) 时功率恒定为 1/L。

**复现必要性**：⭐⭐ **可选**。这主要验证波形合成过程的约束满足情况，属于工程验证类实验，若对合成流程有信心可跳过。

### 实验2b：合成损失（Fig.5）

**目的**：评估 SDR（半正定松弛）引入的松弛间隙。由于原问题 P 是 NP-hard 的，论文对其进行了松弛得到 P̃，然后从松弛解 Rs 中合成实际波形 s*。此实验量化**从 Rs 到 s* 的性能损失**。

**做法**：扫描 ρ 值，对比 f(Rs)（协方差矩阵的目标值，虚线）与 f(s*)（合成波形的目标值，实线）。

**衡量指标**：合成损失 = f(Rs) − f(s*)。

**期望结果**：最大损失 < 0.1，且 ρ 对目标值影响很小（曲线近似直线）。

**复现必要性**：⭐⭐ **可选**。类似实验2a，属于工程验证类。证明随机化合成方案对性能损失影响不大。

### 实验3：SINR统计特性与分布（Table I + Fig.6/7）

**目的**：这是论文**最核心的对比实验**。通过蒙特卡洛仿真对比 PRD 准则与文献[17] 准则设计的波形在随机TIR下的SINR分布差异，直观展示 PRD 的概率鲁棒性。

**做法**：
1. 分别用 PRD 和 [17] 设计波形
2. 随机生成 1000 个 TIR 样本 t ~ CN(t₀, Rt)
3. 计算每个样本的输出 SINR：γ = t^H · M · t
4. 绘制 SINR 分布直方图（Fig.6/7），统计 E[γ]、D[γ]、f 值（Table I）

**衡量指标**（Table I 的三列）：

| 指标 | 含义 | PRD的优势 |
|:---|:---|:---|
| E[γ] | SINR期望 | PRD略低（牺牲期望） |
| D[γ] | SINR标准差 | **PRD显著更低**（方差受控） |
| f = (E[γ]−γ₀) / D[γ] | PRD指标 | **PRD显著更高** |

> [!IMPORTANT]
> 核心发现：在高SINR下，[17]方法的 E[γ] 和 D[γ] 以**相同比例**随 σ² 降低而增长，导致 f 值提升有限。而PRD准则在增大 E[γ] 的同时**有效抑制** D[γ] 的增长，使 f 值大幅领先。
>
> 在低SINR下，两种方法表现一致——这也与论文预期符合。

**性能提升量化**（基于目标1的诊断输出）：

| σ²(dB) | SINR情况 | [17]的 f | PRD的 f | PRD提升 |
|:---|:---|:---|:---|:---|
| −10 | 高 | 0.91 | 2.28 | **+150%** |
| −5 | 高 | 0.67 | 1.24 | **+85%** |
| 0 | 低 | −0.11 | −0.11 | 基本相同 |
| 5 | 低 | −2.57 | −2.70 | 基本相同 |

**复现必要性**：⭐⭐⭐⭐⭐ **必须复现**。这是论文的核心性能验证实验，**已完成复现**。

### 实验4：波形PSD对比（Fig.8/9）

**目的**：从**频域角度**解析两种方法设计波形的本质差异，回答"为什么PRD更鲁棒"的问题。

**做法**：对比设计波形的功率谱密度（PSD）与目标 TIR 的 PSD。

**关键发现**：
- **[17]方法**：波形PSD始终在目标PSD最大峰处形成**单一主峰**，不随 σ² 变化 → 将所有能量集中在"最赚"的频点，贪婪策略
- **PRD方法（高SINR）**：波形PSD趋于**覆盖目标PSD的完整结构**（多峰、宽频） → 在多个频点分散能量，即使某个频点的TIR偏离预期，整体性能仍有保障
- **PRD方法（低SINR）**：与[17]几乎相同 → 资源有限时无法分散，只能集中

**复现必要性**：⭐⭐⭐⭐ **建议复现**。这个实验提供了直觉上的理解，帮助解释PRD准则为何有效。**已完成复现**。

### ❌ 实验5：稳定检测概率曲线（Fig.10）— 未实现

**目的**：这是论文中**最终的定量性能评估实验**。前面的实验都是从 f 值、分布形态、PSD 等间接指标来对比，而 Fig.10 直接回答了一个最实际的问题：**"在给定噪声水平下，检测到目标的概率有多高？"**

**稳定检测概率的定义**：

```
Psd = N_EX / N_TO
```

其中：
- N_TO = 蒙特卡洛总试验次数
- N_EX = 事件 {γ(s) ≥ γ₀} 发生的次数（即输出SINR超过阈值的次数）

通俗地说：**随机抽取一千个可能的目标状态，用设计好的波形去检测，有多大比例能达到期望的检测性能。**

**实验设置**：
- 横轴：噪声功率 σ²（从大到小扫描，越小意味着越高SINR）
- 纵轴：稳定检测概率 Psd
- 曲线：PRD（蓝色）vs [17]（红色），分别对目标1和目标2各画一张图

**论文的期望结果**：
- **σ² 较大**（低SINR）：两条曲线重合，Psd 都很低
- **σ² 减小**（进入高SINR）：PRD 曲线**更快上升**，获得更高的 Psd
- **σ² 很小**（深度高SINR）：两者都趋近于 1，但 PRD 更早到达

**物理意义**：PRD 设计的波形能在**更宽的噪声范围内**保持高检测概率，体现了"概率鲁棒性"的名称由来。

**复现必要性**：⭐⭐⭐⭐⭐ **强烈建议复现**。这是论文中唯一直接从**检测概率**角度量化性能提升的实验，是全文的**收官实验**。而且实现简单——数据在 exp3a 中已有（蒙特卡洛的 SINR 样本），只需统计超过阈值的比例并画曲线即可。
